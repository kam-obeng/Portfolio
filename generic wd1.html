Technical Report

Project Overview

My first software project was a personal portfolio website, created as part of my final web development assignment. This project was a significant milestone in my learning journey as a software engineering student, allowing me to apply my knowledge of HTML, CSS, and JavaScript in a practical setting. The main goal of this website was to introduce myself professionally, showcase my work, and experiment with fundamental web development concepts.

Design and Features

The website's design and layout were intentionally kept simple and consistent across all pages. I chose a colour scheme consisting of dark red, black, grey, and white to give the website a bold yet professional appearance. The navigation bar was placed at the top of the page to enhance usability and provide a sleek design. Additionally, I included an aside section featuring three social media iconsâ€”Indeed, Instagram, and GitHub. Clicking on any of these icons would redirect the user to the corresponding platform, demonstrating my ability to integrate hyperlinks effectively.

Dropdown Menus and Layout

One of the key features of the website was the dropdown menu incorporated into the navigation links. This was implemented using HTML and CSS to provide an organised and accessible way to navigate different sections of the site. Another design choice I made was to use columns on two of the pages to display text. This was done to show versatility in text presentation and layout structuring. By experimenting with different text arrangements, I was able to demonstrate my understanding of CSS layout techniques such as flexbox and grid.

Development Process

From a coding perspective, the project was relatively straightforward. HTML and CSS concepts were easy for me to grasp, and the process of implementing the design elements was smooth. However, I encountered some challenges, particularly in ensuring consistency across different screen sizes. To address this, I implemented media queries, creating separate stylesheets for mobile and laptop screens. This helped improve the website's responsiveness, ensuring a better user experience across various devices.

Conclusion

Looking back, this project was a great learning experience that laid the foundation for my web development skills. It not only reinforced my understanding of front-end technologies but also encouraged me to explore more advanced concepts such as JavaScript interactivity, backend integration, and database management. Overall, this website reflected my personality through its design choices while serving as a testament to my growing skills in web development.
