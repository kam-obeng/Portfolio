Mobile App Assignment 2

Basic System Requirements

1. Registration

Students can register with the following details:

  • Full Name (Done)
  • Date of Birth (Done)
  • Secondary School Email Address (Done)
  • Date Created (Done)
  • Date Updated (Done)

Students can update their profile details. (Done)

2. Login

Students can log in using their secondary school email address. (Done)

3. Posts

Students can create, edit, update, and view posts. (Done)

They can also comment on posts created by their peers. (Done)

4. Admin

Admins can manage students by performing the following actions:

  • Delete a student. (Done)
  • Manually add a new student. (Done)
  • Update a student’s password. (Done)
  • Update a student’s information. (Done)

Enhanced Features

  • Visually appealing user interface. (Done)
  • Polls and surveys functionality. (Done)
  • Multilingual support (Currently available in French). (Done)
  • Like and dislike feature on comments. (Done)
  • Wikipedia web view added to the menu page. (Done)

Conclusion

The project successfully implemented core social media functionalities for the school, meeting all requirements and adding extra features.

Like and comment functionalities have enhanced user interaction.

Potential improvements include better search functionality, advanced error validation, and more profile customisation options.

Through this project, a deeper understanding of user authentication, CRUD operations, and error handling was gained.
