Proficiency in software tools that implement databases

Proficiency in database tools like XAMPP, SQL Workbench, and MySQL is crucial for efficient database management. XAMPP provides a local server environment with Apache and MySQL, making it ideal for testing and development. SQL Workbench offers an intuitive interface for designing, querying, and optimizing MySQL databases. Additionally, tools like Oracle SQL Developer and Microsoft SSMS support advanced database administration, performance tuning, and data security. Mastering these tools enables efficient database design, query execution, and seamless integration into applications.

Object-Relational Database System for Organisational Management

This project involves the design, implementation, and automation of an object-relational database system using PL/SQL to manage organisational data efficiently. The database comprises entities such as teams, guides, guests, programs, and practices, with structured relationships managed through entity-relationship diagrams (ERDs), foreign keys, and constraints. The system integrates stored procedures, triggers, cursors, and user-defined functions to automate data validation, employment status determination, and record insertion. Advanced SQL queries facilitate data analysis, such as retrieving program costs, counting guest participation, and filtering results by specific conditions. The database architecture supports scalability and automation, enforcing integrity through triggers that prevent invalid data entries (e.g., future dates for birth records). The project also includes test plans ensuring functionality, performance, and compliance with organisational requirements. Overall, this system provides a robust and automated solution for managing complex data relationships in an organisational setting.
