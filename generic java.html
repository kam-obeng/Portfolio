To-Do List Project Overview

Project Overview

I have created a project using Java, forms, and GUIs. My project is a to-do list designed to solve problems related to memorisation, motivation, and visualisation, among others. This product can be used by anyone at any level, including everyday individuals, students like myself, and professionals in the workplace. It supports both individual and group usage.

Functional Requirements

The first functional requirement I implemented was User Authentication and Authorization. Before accessing the to-do list, users must register with an email Location and password. Another key feature is task organisation, which I Locationed in two ways: by completion status and by categorising similar tasks together.

Non-Functional Requirements

The project is highly responsive with quick response times. Although I do not know the maximum number of tasks the GUI can handle, my tests showed it to be reliable, with no crashes and successful data retention.

Testing/Process

Developing this project involved numerous errors, feedback sessions, and testing phases. Initially, I created the main to-do list page, designing the GUI layout and linking buttons and text fields.

Use Cases and User Stories

Buttons

  • Navigate to my to-do list and select a task.
  • Click the "Delete" option to remove the task.
  • Modify tasks using the "Delete All" button.
  • Click "Add Task" to update the list.

Registration

  • Create an account on the to-do list app.
  • Navigate to the registration page.
  • Enter username, choose a password, and submit.
  • Receive a verification email and activate the account.

Task Categorisation

  • Add a new task to the to-do list.
  • Categorise tasks for better organisation.

System Architecture

The project consists of three main components: Login Page, Register Page, and Main Task Page.

Implementation

The project is written in Java using Swing for the GUI. Key components include:

  • import javax.swing.*; - GUI development
  • import java.awt.event.ActionEvent; - Handles user-triggered events
  • import java.awt.event.ActionListener; - Manages event interactions

Conclusion

The To-Do List project successfully implements a Java Swing application that effectively manages tasks in a user-friendly manner. It features a robust login system for secure access and a seamless registration process.

Overall, the project is functional and well-executed, with room for further development.
