Graphical User Interface for Task Management

This project utilises Java for the development of its form and graphical user interfaces (GUIs). It is designed as a to-do list application aimed at addressing challenges related to memorisation, motivation, visualisation, and more. This versatile product caters to a wide audience, encompassing individuals of various backgrounds and proficiency levels. The target demographic ranges from everyday civilians to students and professionals alike.

The primary focus of the project was on enhancing visualisation. This was achieved through the implementation of a streamlined and aesthetically pleasing GUI design, complemented by judicious use of colours and imagery to enhance its presentation. The interface provides users with a comprehensive view of their tasks, presented in an orderly list format. Furthermore, the project features a select number of clearly labelled buttons such as "Add Task," "Edit Task," "Delete Task," and "Delete All Tasks."
